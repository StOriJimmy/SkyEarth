张永军教授团队科研论文荣获2023年度E.H.汤普森奖（E H Thompson Award）

（通讯员：姚永祥）日前，在英国约克National Stem中心闭幕的英国遥感与摄影测量学会（The Remote Sensing and Photogrammetry Society）2024年年会上，我院张永军教授团队2021级硕士研究生杨吴鹏（第一作者），指导教师张永军教授、姚永祥老师（共同通讯作者）等完成的论文“Weak texture remote sensing image matching based on hybrid domain features and adaptive description method”荣获英国遥感与摄影测量学会2023年度E.H.汤普森奖（E H Thompson Award）。

获奖证书

论文从图像特征提取、描述符设计等方面分析了弱纹理遥感影像匹配面临的挑战，提出了一种新颖的混合域特征与自适应描述方法，旨在解决低反射率、像素高邻域相似性以及区域间差异不明显等导致的匹配失败问题。该方法提出了联合空间域与频率域尺度空间的新特征提取器；设计了一种自适应邻域的对数极坐标描述符，根据对数归一化强度值进行自适应邻域窗口计算。该方法在包含沙漠、森林、水域、冰雪、阴影等弱纹理场景的典型遥感影像上进行了验证，匹配性能卓越，具有广阔的应用前景。

文献引用格式：Yang, W., Yao, Y., Zhang, Y., & Wan, Y. (2023). Weak texture remote sensing image matching based on hybrid domain features and adaptive description method. The Photogrammetric Record, 38(184), 537-562. https://doi.org/10.1111/phor.12464.

英国遥感与摄影测量学会（The Remote Sensing and Photogrammetry Society）每年从《The Photogrammetric Record》期刊上一年度发表的所有论文中评选出一篇最佳论文，并冠名E H Thompson Award奖。E H Thompson Award由The Photogrammetric Record期刊颁发，旨在奖励《The Photogrammetric Record》期刊上发表的最具原创性和价值的论文，以促进摄影测量学领域的研究和发展。
