《中国测绘》专访张永军教授

by admin Dec 29, 2019

今年8月2日，CCTV-10科教频道《创新一线》栏目播出专题片《遥感先锋》，系统介绍了武汉大学遥感信息工程学院张祖勋院士及张永军教授团队完成的航空航天遥感影像摄影测量网格处理关键技术，其中对该项技术的研发历程、基本原理、主要应用等进行了介绍，让这项有重大应用价值的创新技术步入更多大众的眼帘。作为团队核心成员、项目负责人的张永军教授长期以来跟随张祖勋院士亲历摄影测量网格处理关键技术的研发过程，并极力推进了该技术的全球推广，本期《中国测绘》杂志专访张永军教授，了解关于摄影测量网格处理关键技术推广应用的更多故事。

课题组科研成果获评国家科技进步二等奖

by admin Jan 08, 2018

1月8日，中共中央、国务院在京隆重举行2017年度国家科学技术奖励大会。武汉大学共4项成果获国家科学技术奖励，其中主持完成2项成果分别获国家自然科学二等奖和国家科技进步二等奖，参与完成的2项成果获国家科技进步二等奖；遥感信息工程学院张永军教授主持完成的“航空航天遥感影像摄影测量网格处理关键技术与应用”获国家科技进步二等奖。党和国家领导人习近平、李克强、张高丽、王沪宁等出席大会并为获奖代表颁奖。
